# Simple-Text-Editor

### Описание
Упрощенный текстовый редактор с поддрежкой базового набора команд:
Left - перемещение курсора влево на одну позицию.
Right - перемещение курсора вправо на одну позицию.
Insert - ввод символа в текущую позицию курсора.
Copy - копирование фрагмента текста, начинающегося в текущей позиции курсора, в буфер обмена.
Cut - вырезание фрагмента текста (аналогично Copy, но с последующим удалением этого текста).
Paste -  вставка содержимого буфера обмена в текущую позицию курсора.

### Запуск теста
Обязательно наличие компилятора g++ с поддержкой стандарта С++17 и выше, а также утилиты make.
Для запуска теста класса, необходимо перейти в папка make-test и вызвать make.
Также запуск теста можно осуществить из папка bin
